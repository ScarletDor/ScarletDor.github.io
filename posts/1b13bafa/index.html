有道云笔记|桌面版|去广告|小笔记
发表于 2020-02-10| 更新于 2020-02-15|笔记电脑软件
字数总计:536|阅读时长: 2 分钟

这几天火喵打算整一个用来全平台写笔记的软件，因为苹果 就一个ipad所以也就是安卓和win了。手机上写一半电脑接着写这种。看到说印象笔记支持md了于是立刻试了一下效果并不好，突然想起同样做笔记生意的网易，果然有道云笔记除了界面不华丽，确实简单好用。但是碍于本喵强迫症，左下角有广告啊，咋办呢~

  1. 看到目录下YoudaoNote\theme\有一个build.xml文件，打开他，搜索ads这个缩写，竟然啥都没有，那就搜一下广告吧，嘿嘿有啦~
    全都删掉删掉~重启有道云笔记！咚！糟糕！打开错误，无法打开了！

  2. 不能删那就修改吧
    将底部广告：

    xml

    里面的161换成0,即：

    xml

    然后去除广告占位符：

    xml

    将PanelAd标签内所有含有Bounds和Margin属性的值全都修改为0，0，0，0：

    xml

    完成啦~

  3. 现在 有道云笔记 6.9.0.0（去广告版）已经上架软件中心啦，懒小伙伴快去下载哦~
