2020，月夜杂记

2020年来临啦，在这里记录下每一天的琐事，期待着新的一天的到来~

春之风

1月

1月1日

说说页面做好啦！以后在这里偶尔发发牢骚啦~

1月2日

今天改了鼠标指针样式，也把部分常用软件的下载页面做好啦~快过年了，不错不错！

1月5日

终于想方设法加上了实验室的页面，再考虑把好看的html搬过来做个关于页面，但是有时候css会冲突，这就很难受了，，话说我从wordpress都搬过来那么wp留着干嘛hhhh

1月6日

今天学到了hexo的markdown文章里添加网易云音乐的外链，方法就是要把网易云上面的外链播放器代码复制过来，然后在它的属性width和height的值上面加个双引号就行了；想要全宽的话写上width=”100%”,另外这里改名为小站心语了，以后只记录小站大事件，说说页面另想办法啦嘿嘿~就这样！晚安！！

1月10日

今天炮姐第三季终于！！！开播啦！！！最爱炮姐的泡泡袜和大长腿了！！！！！

1月13日

今天把这个新页面做出来了，以后这里写写心情~置顶置顶！！！，之前的小站心语页面删除！相册页面删除！重新调整目录！真棒！！！

1月16日

记一次被坑：
  昨天部署博客，在页面底端加上了 与火喵 在线聊天.，谁曾想，网站deploy之后一看，页面没有更新，我就想也许是没部署上吧，于是去coding一看还真是，于是就立即重新部署。结果发生了这种情况：

没看错，说我20002错误了…我就很气怪，我也没做什么啊，但是这怎么来的错误呢，试了一晚上都没成，而且coding每小时只能部署十次，失败了十次就睡觉去了。

今天晚上回来，打算重新部署一下试一试，在想要是不行的话就回复三天前的备份，结果，，成功了？？？原来是coding的bug么？？？

1月17日

今天是小年~开心！睡不着觉啦！想看好多的动画片！
言归正传，今天记录一下Markdown的语法
除了一些简单的，今天记录一下消息提示框！

markdown

!> 文字 表示黄色警告框
i> 文字 表示蓝色信息框
@> 文字 表示银色引用框
x> 文字 表示红色错误框
√> 文字 表示绿色成功框

试了一下竟然不是，应改为：

html
default
primary
success
warning
danger
info

1月18日

今天和妈妈咪去看电影啦~
紫罗兰永恒花园，京阿尼火灾前最后一部作品，据说四月份还有新剧场版，可是我tv还没看完呢~
电影过后就吃的鸡公煲，还有章鱼小丸子啦，关东煮啦各种好吃的哈哈哈开心死了

小记：今天加上了谷歌广告！真棒！！！

1月21日

今天成功加上谷歌广告啦！考虑了一下就不在主页和文章内加广告了，现在只有侧边栏和移动设备的边缘有广告~
期待月入广告费1w的那一天，我会努力更博客的！！

1月24日

农历最后一天了,买个游戏送给自己，看了看还是买只老鼠吧，毕竟也是鼠年嘛~

3月5日

最近把博客主页改成了三栏，而且也成功的加上了小动态的侧边栏，目的是可以随时的发说说，但是没想到可以直接修改一下css然后放到主页侧边栏。
但是这个小东西和日记，留言板功能有重叠，所以也都各自定了一下位：

  • 留言板：较为正式的留言板，给各位访客们留下足迹，一般自己不会主动发言；
  • 小动态：非正式的留言框，目的是随时随地留言，一般自己发言；
  • 日记： 记录生活中有意义的事，值得纪念的事情。
